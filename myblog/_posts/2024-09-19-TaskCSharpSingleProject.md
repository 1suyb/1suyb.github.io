# [내일배움캠프] C# 미니 프로젝트
## 요구사항
### 필수기능
1. 게임시작화면
	1. 간단한 소개
	2. 입력가능한 메뉴 보여주기
	3. 입력받기
2. player 상태 표시
	1. Player 속성
		1. 레벨
		2. 이름
		3. 직업
		4. 공격력
		5. 방어력
		6. 체력 
		7. gold
	2. 속성 보여주기
	3. 입력받기
		1. 0을 눌러 돌아가기
3. 인벤토리
	1. 행동
		1. 장착중인 아이템 보여주기
		2. 장비 아이템 장착하기
			1. 아이템 목록 보여주기
			2. 입력받기
			3. 입력받은 숫자의 장비 장착하기
			4. 정보 반영하기
		3. 돌아가기
	2. 속성
		1. 소지중인 아이템
		2. 골드
4. 상점
	1. 행동
		1. 보유중인 골드 표시
		2. 아이템 정보 가격 표시
		3. 구매 완료한 아이템은 구매 완료 표시
		4. 아이템구매
			1. 아이템 목록 표시
			2. 입력받기
			3. 구매
				1. 골드가 충분하다면
					1. 골드 감소
					2. 인벤토리에 추가
					3. 구매완료표시
				2. 골드가 충분하지 않다면
					1. 골드가 부족합니다 출력
	2. 속성
		1. 판매하는 아이템 목록
